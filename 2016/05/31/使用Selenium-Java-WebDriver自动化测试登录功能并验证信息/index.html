使用Selenium Java WebDriver自动化测试登录功能并验证信息

  • WebDriver
  • java
  • selenium
  • software test

在测试中我们经常会遇到一个问题，那就是如果我们测试一个登录功能时，需要验证用户名和密码是否正确，一两个用户名密码倒是无所谓，我们可以手动操作，然而如果遇到几百个用户名密码需要验证我们还是用手动输入验证的方式的话，那就太有毅力了。所以为了解决这样的问题，才衍生出了“自动化测试”。
在这里，我们就用Selenium进行自动化测试。

一、首先，我们可以选择一种浏览器，Chrome或者是Firefox，我们这里是选择Chrome，因为Firefox在启动时很慢，个人感觉挺浪费时间的，虽然他的代码写起来比较简单，我们还是比较注重性能的，哈哈。

二、我们的用户名等一些数据都放在了一个CSV里面。我们要对CSV进行操作，读取数据，并进行自动化测试。思路如下：
（1）获取登陆页面用户名和密码的文本框对象
（2）对用户名和密码的文本框对象进行赋值操作，并触发登录提交按钮（其中赋值的内容，通过java操作CSV文件进行批量操作）
（3）对是否能登录进行判断
（4）对登录后的页面进行元素获取，获取关键的邮箱信息，如果能获取到邮箱信息则登陆成功，否则登录失败
（5）对于登录成功的页面，将获取的邮箱信息和CSV中的信息匹配，相等则信息也正确，否则信息错误
