使用Junit和Eclemma进行软件测试

  • Eclemma
  • Junit
  • software test

最近做项目越来约发现软件测试的重要性，之前感觉自己软件测试的理论还是比较好的，同时也在开发中经常使用Junit进行单元测试，也同时用过JMeter做过压力测试，但是对于白盒测试中的路径覆盖，或者对于自动化测试方面都没有太多接触，所以最近就试着学习了一下，自己也尝试了一下。

下面我们就根据一下小问题来做路径覆盖测试：
【问题】：
输入三角形的三条边a,b,c，设计一个函数判断三角形的类型是equilateral,isosceles,还是scalene，并对函数进行路径覆盖测试

【着手开始解决】：
1、新建项目
打开Eclipse —> File —> New —> Java Project
2、在Eclipse中安装Junit 4.12和Hamcrest 1.3
(1) 下载jar包： junit-4.12.jar hamcrest-all-1.3.jar
(2) 在项目中新建lib目录，并将下载的两个主要jar放入，Build Path加入到项目依赖中
(3) 新建类Calc.java和JunitTest.java用于测试Junit和hamcrest是否安装成功
