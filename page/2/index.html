写在校庆的那天深夜

    打开手机，深夜时刻03:27，不是起得太早，而是久久没有睡去。耳朵里塞着何洁的《你一定要幸福》，感觉到这样的心，曾经寂寞过，也曾如此委屈过。每当听到这首歌，总会想起一个不错的朋友，看朋友圈的说说，这个时候估计应该是在台湾和妈妈在开心的旅游吧。真心为她高兴，因为我知道，微笑的她才是真的她。

    今天是10月3日，昨天是我校天津大学的120周年的校庆，盛大而又隆重。早上去老校区转悠了一会儿，又赶回新校区感受一下氛围，很可惜的是没有吃到校庆丰盛的免费午餐，也没有领到只属于天大校友的纪念品，然而心里还是很欢喜，很兴奋，因为这是一个普天同庆的日子。因为我知道，今天的天很蓝，蓝得怎么这么不合逻辑……

    左手Ctrl和S的组合键习惯的保存了一下，眼神稍微走了下光，看到了金老师前几天给我的研究僧第一篇论文，叫《Learning Deep Representations for Graph Clustering》。这对于大学不曾跟老师做过学术研究，只是倒弄JavaEE项目的我来说，可能有点吃力，因为连翻译过来的一些专业名词我都不懂。然而，不管怎样，和伙伴们从1895咖啡厅玩完桌游回来，便静下心来看了Introduction部分。
满纸的注释和翻译，应该连鬼都能看出我的英语有多烂吧。

    桌上的钥匙，其中有两把是自己公司的钥匙。在我看来，这是我未来的事业，一个希望和曹共同经营的事业。我们约定过的，5年天津的打拼和经营，经营属于我们自己的事业。
只能说，在我们自己看来，我们还很稚嫩，不懂的东西太多，现在的我们最好的就是沉淀自己，保持一个永远好奇的心，掌握好自己的技能。总结为一句话就是，“人笨多读书，我们还是太年轻”。

    很喜欢曹曾经写给团队的一句话，他是这样说的：
公司注册只是介绍词，拉开帷幕之前，我们必须编造一段美丽的故事结局，创业期间铭记故事结局，故事主人公都可以换，结局不能换，沉醉在故事插曲不可怕，可怕插曲过后咱们忘了故事美好的结局。
这句话是我们自我分析性格时，他思考出来的一句话。这句话很触动我，我感觉说得很实在，也说得很理性。我是经历过创业的，虽然我的年龄在别人眼里还是个一无用处的小屁孩。然而，不可否认的事，之前一年零三个月的创业经历让我兴奋过，失望过，绝望过……让我饱受了抉择的艰难和痛苦，也从中让我备受触动。
它教会了我什么叫浮躁，什么叫迷茫，什么叫诚信……一切的一切，刻在我的心里，只能说，感谢曾经遇到的你们，教会我成长，教会我自强，教会我在社会上摸爬滚打。

    看着到哪里都带着的照片，忽然感慨，一年半了，怎么能受得了我，怎么受得了坏毛病一身的我呢！嘿嘿，真的很感谢宝贝竟然还跟我这个在你眼里不大靠谱的小屁孩在一起。还记得，大约是年前团购的艺术照，收到之后就被我直接都“收藏”了，钱包里，桌面里…..随处看到了你的身影，把周围的狗狗们虐得嗷嗷叫，嘿嘿，应该说我感到了一种夫妻的感觉，现在的我，感到很幸福，因为现在的我，有你陪伴 O(∩_∩)O

    2015年10月，研究僧生涯的开始。
    为了家人的幸福，我，一直在努力……

TCP Scoket

今天我们讨论的是关于TCP/IP网络中两个进程之间相互作用的主机模式，即Client/Server Model。
CLient和Server端典型场景如下：
1、Server Startup
2、Client Startup,then connects to the Server
3、Client send a request to Server
4、Server process the request,and sends a reply back to the Client
5、Loop 3 and 4 steps…(data transmit)
6、Client close its end of the connection,send a end-of-file notification to Server
7、Server close its end of the connection (Terminate or Wait for a new Client connection)

Linux下tomcat日志文件的分割备份

如果 catalina.out 日志达到 2GB 大小的时候，Tomcat 因为缓存问题，便没有办法继续输出日志了。
为了避免这种情况，你需要及时对 catalina.out 进行归档。
实际上 Tomcat 自身也会对日志内容进行归档，但是归档后，Tomcat 不会清理 catalina.out 的内容，这将会导致 catalina.out 越来越大。
